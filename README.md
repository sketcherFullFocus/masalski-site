# Masalski — Residences & Promenades

Лендинг по продаже коммерческих помещений и апартаментов в отреставрированном
доме князей Масальских (памятник архитектуры XVIII в., центр Гродно).
Первый этаж — бутики и рестораны, верхние этажи — апартаменты для краткосрочной аренды.

**Живой сайт:** https://sketcherfullfocus.github.io/masalski-site/

## Стек

Статический сайт без сборки и зависимостей: один `index.html` (встроенные `<style>`
и `<script>`), шрифты и изображения — в `assets/`. Никакого React, npm или CDN.
Открывается двойным кликом по `index.html`.

- ✅ Десктоп
- ✅ Мобильная и планшетная версия (адаптив)
- ✅ Анимации: появление секций при скролле, Ken Burns/параллакс фото,
  вход текста героя, липкая шапка, плавная смена карточек каталога
- ✅ Интерактивный каталог помещений (фильтр Все / Коммерция / Апартаменты) — на чистом JS

## Структура

```
masalski-site/
├── index.html        # вся страница: HTML + CSS + JS
├── assets/
│   ├── hero.jpg              # герой — фасад
│   ├── about.webp           # блок «О проекте»
│   ├── dvor.jpg             # внутренний двор
│   ├── format-boutique.jpg  # формат «Бутик»
│   ├── format-restaurant.jpg# формат «Ресторан/кафе»
│   ├── format-apartments.jpg# формат «Апартаменты»
│   └── *.woff2              # шрифты Cormorant Garamond + Montserrat (self-hosted)
└── README.md
```

Секции (сверху вниз): шапка · герой · интро-цифры · о проекте · внутренний двор ·
форматы · выбор помещения (каталог с фильтром) · управление · инвестиции ·
локация (карта) · форма-CTA · футер.

## Как редактировать

- **Тексты и цифры** — прямо в `index.html` (поиском по нужному слову).
- **Каталог помещений** — массив объектов в `<script>` внизу `index.html`
  (рядом с разметкой каталога), там же — цены, площади, доходность.
- **Фото** — заменить файлы в `assets/` (имена сохранять) либо поправить ссылки в `index.html`.

## Локальный просмотр

Просто открыть `index.html` в браузере. Либо поднять статический сервер:

```bash
npx serve .        # или: python -m http.server
```

## Хостинг

Сайт публикуется через **GitHub Pages** (ветка `main`, корень репозитория).
После каждого `git push` в `main` живая версия обновляется автоматически за ~1–2 минуты.

## Совместная работа

Проект — это содержимое **этого репозитория** (`index.html` + `assets/` + `README.md`).
Больше ничего для работы не нужно: ни сборки, ни зависимостей.

**Подключить человека к проекту:**
1. GitHub → репозиторий → **Settings → Collaborators → Add people** → ввести его GitHub-логин.

**Рабочий цикл для участника:**
```bash
git clone https://github.com/sketcherFullFocus/masalski-site.git
cd masalski-site
# правит index.html / assets, смотрит локально (двойной клик по index.html или npx serve .)
git add .
git commit -m "что изменил"
git push                       # через ~1–2 мин live-сайт обновится
```

Вся история правок хранится в git — можно вернуться к любой прошлой версии.
