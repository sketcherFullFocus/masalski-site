# Masalski — Residences & Promenades

Лендинг по продаже коммерческих помещений и апартаментов в отреставрированном
доме князей Масальских (памятник архитектуры XVIII в., центр Гродно).
Первый этаж — бутики и рестораны, верхние этажи — апартаменты для краткосрочной аренды.

## Статус

- ✅ Десктопная версия готова
- ⛔️ **Мобильной версии нет** — в стилях ни одного `@media`. Адаптив нужно сделать с нуля.
- ⚠️ Кастомный курсор (`cursor: none`) на тач-экранах не работает — отключить на мобилке.

## Стек

Статический сайт без сборки: один `index.html` (встроенные `<style>` и `<script>`),
шрифты — Google Fonts. Зависимостей и npm нет. Открывается двойным кликом.

## Структура

```
masalski-site/
├── index.html        # вся страница (HTML + CSS + JS)
├── images/           # 7 изображений
│   ├── img-1.jpg     # герой — фасад (используется 3×)
│   ├── img-2.jpg     # блок «О проекте» — фасад днём
│   ├── img-3.webp    # внутренний двор, ночь («Сердце Masalski»)
│   ├── img-4.jpg     # карточка формата помещения
│   ├── img-5.jpg     # формат — фото 1
│   ├── img-6.jpg     # формат — фото 2
│   └── img-7.jpg     # карточка формата помещения
```

Секции страницы (сверху вниз): шапка · герой · о проекте · внутренний двор ·
форматы помещений · выбор помещения (табы) · управление арендой · инвестиции ·
локация · форма-CTA · футер.
